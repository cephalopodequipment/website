Public key converter

Cosmos SDK v0.43 (bech32-encoded) -> Cosmos SDK v0.44 (base64-encoded) public key converter.

- This will only work for regular public keys.
- Multisig keys have to be recreated using each key and the CLI.
- Wallet addresses cannot be converted.

This tool is provided AS IS. Use it at your own risk. Using the CLI to recover a pubkey is the safer option.
